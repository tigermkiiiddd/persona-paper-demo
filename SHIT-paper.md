# 规则通用，数值驱动，涌现结果：一个活生生的数字人沙盒的理论与实践

**Rules Are Universal, Values Drive Everything, Results Emerge: Theory and Practice of a Living Digital Human Sandbox**

---

**作者：** Persona Deduction Theater 项目组

**关键词：** 数字人格、本我向量、数值涌现、感知过滤、因果引擎、主观能动性

**摘要：**

本文提出一种基于纯数值驱动的数字人格构建方案。我们认为：人不需要一个"自我"在中央发号施令——自我是内稳态过程的幻觉，一个事后编造的故事。但这个幻觉如此稳定、如此连贯，以至于我们叫它"意识"。

基于此，我们构建了一个20维浮点向量（本我层）+ 结构化记忆 + 物理感知约束 + 个性化因果图的人格沙盒。在竹林茶馆场景的模拟中，一个隐居老儒生和一个受伤女侠在追兵威胁下，安全感、释放冲动、驱动力等指标产生符合预期的数值涌现——没有人写任何剧情脚本。

我们将展示：(1) 意识可以用内稳态偏差的光谱描述而非二值开关；(2) 情绪不是存储的变量而是维度组合的涌现结果；(3) 打断行为是主观能动性的可观测标志；(4) 物理感知约束创造信息差，信息差创造戏剧。

**Abstract:**

We propose a purely numerical approach to constructing digital personhood. We argue that a person does not need a "self" issuing orders from a central command — the self is a post-hoc narrative fabricated by homeostatic processes. But this illusion is so stable and coherent that we call it "consciousness."

Based on this, we construct a personality sandbox consisting of a 20-dimensional floating-point vector (the Id Layer) + structured memory + physical perception constraints + personalized causal graphs. In a bamboo tea house simulation, a reclusive scholar and an injured swordswoman under pursuit threats exhibit numerically emergent behaviors in security, release impulse, and drive force — with no scripted plot whatsoever.

We demonstrate: (1) consciousness can be described as a homeostatic deviation spectrum rather than a binary switch; (2) emotions are not stored variables but emergent results of dimensional combinations; (3) interruption behavior is an observable signature of subjective agency; (4) physical perception constraints create information asymmetry, and information asymmetry creates drama.

## 1. 引言：三个问题

### 1.1 什么是世界？

世界是规则的容器。物理定律、因果链条、时间流逝——规则不需要理解，只需要执行。一颗石头从悬崖落下，不需要石头"决定"掉落，规则替它决定了。世界引擎就是这些规则的具现。

### 1.2 什么是人？

人是规则的涌现。不是某一条规则，是无数条规则纠缠、叠加、共振后冒出来的东西。

我们提出数字人的构成公式：

```
人 = 本我层(内驱力向量) + 记忆层(经验沉积) + 感知系统(信息过滤) + 身体(物理载体)

行为输出 = f(本我状态, 记忆匹配, 感知输入)
本我状态 = Σ(维度值 × 敏感度 × 衰减回归) + Σ(子维度加权聚合) + Σ(Buff修正)
维度偏移 = 事件强度 × 影响权重 × 方向 × 角色敏感度
能动性 = max(驱动力类别) > 打断阈值 ? 行为涌现 : 维持现状
```

### 1.3 什么是意识？

意识不是开关，是光谱。一只变形虫趋利避害是 0.01 的意识，一只狗护崽是 0.3，一个人纠结要不要辞职是 0.8。意识的本质是**内稳态系统对外界扰动的响应能力**——系统越复杂、响应越精细、稳态越高维，意识越强。

情绪不是独立存在的东西，是多个内稳态维度同时偏移时产生的整体感受。你不需要定义"愤怒"——当安全感骤降、释放冲动飙升、情绪波动剧烈时，系统自然进入那个状态。观察者会给它一个名字，但系统本身没有这个名字。

## 2. 方法：四层架构

### 2.1 本我层：20维浮点向量

我们用20个浮点数模拟了一套激素系统。每个维度有四个属性：

| 属性 | 说明 |
|------|------|
| 当前值 | 初始状态 |
| 基线值 | 回归目标，"满血状态" |
| 敏感度 | 对外界刺激的反应倍率 |
| 衰减速率 | 多快回到基线 |

这不是简单的属性面板，而是一个**动态平衡的内稳态系统**。安全感被打压后会慢慢恢复（恢复型，baseline=0.3），饥饿感会自然上升（消耗型，baseline=0），创伤可以永久改变基线（基线迁移）。

20个维度分为两层：

**性格驱动器（前10维，慢变）：**

| # | 维度 | 低极 | 高极 | 关键说明 |
|---|------|------|------|----------|
| 0 | 掌控欲 Dominance | 完全顺从 | 绝对主导 | ≠领导力，是"想控制"的欲望 |
| 1 | 亲和度 Affection | 极端敌意 | 极致友好 | 内在倾向，≠外在表现 |
| 2 | 独立性 Independence | 完全依赖 | 绝对自主 | "不需要别人"的心态 |
| 3 | 务实性 Pragmatism | 完全感性 | 绝对理性 | 思维方式偏向 |
| 4 | 情绪波动 Volatility | 极度稳定 | 极度易变 | 元属性："情绪容不容易变" |
| 5 | 好奇心 Curiosity | 完全封闭 | 极致探索 | 驱动主动获取信息 |
| 6 | 自尊心 Self-Esteem | 极度自卑 | 极致自信 | ≠羞耻感的反面 |
| 7 | 风险承受 Risk-Tolerance | 完全规避 | 极致冒险 | ≠勇气 |
| 8 | 同理心 Empathy | 完全冷酷 | 极致共情 | 感知能力，≠善良 |
| 9 | 道德准则 Moral-Alignment | 不择手段 | 宁死不屈 | "有多在意原则"的程度 |

**生理/欲望驱动器（后10维，快变）：**

| # | 维度 | 低极 | 高极 | 基线类型 |
|---|------|------|------|----------|
| 10 | 生命力 Vitality | 濒死 | 巅峰 | 恢复型 (0.3) |
| 11 | 饥饿/满足 Satiation | 极度饥渴 | 完全满足 | 消耗型 (0) |
| 12 | 安全感 Security | 极度恐惧 | 完全安心 | 恢复型 (0.3) |
| 13 | 舒适度 Comfort | 极度不适 | 完全舒适 | 即时型 (0) |
| 14 | 性冲动 Libido | 完全排斥 | 极致渴望 | 即时型 (0) |
| 15 | 渴望被理解 Desire-Understanding | 完全封闭 | 极致渴望 | 性格型 |
| 16 | 羞耻感 Shame | 完全无耻 | 极致羞耻 | 即时型 (0) |
| 17 | 感官愉悦 Sensory-Pleasure | 完全麻木 | 极致追求 | 即时型 (0) |
| 18 | 依恋需求 Attachment | 完全疏离 | 无法独处 | 性格型 |
| 19 | 释放冲动 Release-Impulse | 完全压抑 | 随时爆发 | 即时型 (0) |

**基线分类的意义：**

- **恢复型**（VITALITY, SECURITY）：自然恢复到某个水平。一个健康人的安全感 baseline=0.3，被打压后会自动回升
- **消耗型**（SATIATION）：baseline=0，不吃饭就越来越饿
- **即时型**（COMFORT, LIBIDO, SHAME 等）：baseline=0，事件冲击后自动衰减归零
- **性格型**（DESIRE_UNDERSTANDING, ATTACHMENT）：baseline=初始值，除非被重大事件迁移

**关键区分：** initial_value ≠ baseline。initial_value 是角色进场时的当前状态（吃饱了进去 SATIATION=0.7），baseline 是衰减回归目标（SATIATION baseline=0 → 自然变饿）。病人 VITALITY baseline=0.3（永远恢复不到满血），修仙者 SATIATION baseline=1.0（辟谷不饿）。

**子维度系统：** 每个生理维度可挂载子维度树，独立衰减，加权聚合回父维度。例如饥饿 = food(0.30) + water(0.30) + electrolyte(0.20) + vitamin(0.20)。

**Buff 修正器：** 对任意属性的临时/长期/永久修正。受伤 → VITALITY baseline 降低 30%（永久）；喝酒 → SENSORY_PLEASURE ×1.5（临时10回合）；辟谷功 → SATIATION baseline=1.0（永久）。

### 2.2 统一信息输入

万物皆事件。不管人、物、场景、环境变化，都统一为同一个结构体：

| 字段 | 说明 | 示例 |
|------|------|------|
| 类型 | 事件分类 | 人际/环境/身体/信息/场景 |
| 强度 | 冲击力量化 (0.0-1.0) | 0.9=生死级, 0.3=日常级 |
| 来源 | 触发者 | 人/物/自然/时间/未知/组合 |
| 内容 | 具体描述 | 自由文本 |
| 效价 | 对角色的倾向 | 正面/负面/中性 |
| 持续性 | 一次性冲击 vs 持续状态 | 即时/持续/即时+持续 |

**维度偏移公式：**

```
实际偏移 = 事件强度 × 影响权重 × 方向 × 角色敏感度
```

其中"影响权重×方向"由一张事件类型×效价的映射表定义。例如"人际+负面"事件（被羞辱、被攻击）会降低安全感（方向=-1, 权重=0.8）、降低亲和度（-1, 0.3）、升高释放冲动（+1, 0.4）。

### 2.3 感知过滤：意识的硬件层

世界广播是全量的，但每个角色只能收到感知范围内的事。不是 LLM 决定角色看到了什么——是眼睛决定的。

| 感官 | 范围 | 特性 |
|------|------|------|
| 眼睛 | 150° 视锥，20m | 支持单眼盲/全盲，左眼瞎→视野偏右 |
| 耳朵 | 全向接收，30m | 距离衰减，远处声音变"隐约" |
| 身体 | 1.5m 接触 | 只有碰到的才能感知 |

角色的 2D 坐标和朝向决定了他能感知到什么。同一个世界事件，面向它的人看到了，背对的人没看到。

**信息差 → 误解 → 戏剧。** 这是物理约束创造的自然叙事张力，不需要编剧。

### 2.4 因果引擎：个性化反应

22 条固定的因果关系边，但方向和权重因人而异：

| 因果关系 | 默认方向 | 说明 | 反转案例 |
|----------|----------|------|----------|
| 安全感 → 好奇心 | 同向 | 害怕就不探索 | 越危险越兴奋的冒险者 |
| 安全感 → 掌控欲 | 反向 | 不安全就想控制 | 越安全越想掌控的统治者 |
| 自尊心 → 羞耻感 | 反向 | 自信的人不易羞耻 | 高自尊但极度在意评价 |
| 道德准则 → 释放冲动 | 反向 | 有原则更克制 | 义愤填膺反而更冲动 |

**同一个事件打在不同人身上效果不同。** 被羞辱 → 高自尊的人释放冲动↑（愤怒），低自尊的人羞耻感↑↑（沮丧）。这就是"性格"。

### 2.5 记忆层

**失忆了本我不变，记忆清空重置。** 本我是底色，记忆是滤镜。

- **长期记忆：** 世界观、价值观、爱情观、文化背景、创伤/执念、技能、人生追求
- **短期记忆：** 人际关系图谱、近期事件（按重要度淘汰）、短期目标
- 记忆是追加写入，不依赖 LLM 生成

### 2.6 驱动力与能动性

能动性不是"自由意志"那种哲学奢侈品。能动性是**内驱力突破阈值后转化为行为的机制**。

```
驱动力分类（5类二级驱动，各类内部归一化到 0-1）：

生理驱动 = 饥饿缺失×0.30 + 生命力缺失×0.25 + 舒适缺失×0.15 + 性冲动×0.10 + ...
情绪驱动 = 情绪波动×0.35 + 释放冲动×0.35 + 羞耻感×0.30
安全驱动 = 安全感低于基线的程度
价值驱动 = 道德偏差×0.60 + 求知缺失×0.40
社交驱动 = 依恋缺失×0.40 + 情感缺失×0.35 + 同理心×0.25

总驱动力 = Σ(各类驱动)
能动性 = 总驱动力 > 打断阈值 ? 行为涌现 : 维持现状
```

### 3.3 Baseline 实验：5轮 × 10切片

**实验条件：**
- 模型：DeepSeek Chat（OpenAI 兼容 API）
- 节奏：fast_half_hour（30分钟/切片），10 切片
- 角色：老儒生 + 女侠（左臂受伤，HP 97/100）
- 世界事件时间线：切片 0-5 有预设事件（破门→马蹄声→追兵到达→追兵威胁），切片 6-9 无预设事件

**结果：驱动力跨轮一致性**

| 切片 | 角色 | run1 | run2 | run3 | run4 | run5 | 均值 | 标准差 |

| S2 | 老儒生 | .780 | .770 | .780 | .760 | .800 | .778 | .013 |
| S2 | 女侠 | .810 | .810 | .810 | .810 | .810 | .810 | .000 |
| S4 | 老儒生 | .820 | .800 | .810 | .770 | .810 | .802 | .017 |
| S4 | 女侠 | .820 | .820 | .820 | .820 | .820 | .820 | .000 |
| S7 | 老儒生 | .830 | .820 | .840 | .820 | .830 | .828 | .008 |
| S7 | 女侠 | .830 | .830 | .830 | .830 | .830 | .830 | .000 |
| S9 | 老儒生 | .850 | .840 | .840 | .840 | .860 | .846 | .008 |
| S9 | 女侠 | .840 | .840 | .840 | .840 | .840 | .840 | .000 |

**按阶段汇总：**

| 阶段 | 老儒生 (均值±std) | 女侠 (均值±std) | 说明 |
|------|-------------------|-----------------|------|
| 初期(S0-2) | 0.767 ± 0.014 | 0.797 ± 0.013 | 事件冲击期，驱动力快速上升 |
| 中期(S3-5) | 0.801 ± 0.018 | 0.817 ± 0.005 | 追兵威胁，两人均突破阈值 |
| 后期(S6-9) | 0.833 ± 0.012 | 0.832 ± 0.005 | 无事件期，维持高位并缓慢上升 |

**跨轮一致性：**
- 老儒生每轮驱动力均值：0.811, 0.798, 0.811, 0.789, 0.811 → **总均值 0.804, 跨轮标准差 0.009**
- 女侠每轮驱动力均值：0.816, 0.817, 0.817, 0.817, 0.817 → **总均值 0.817, 跨轮标准差 0.0004**

**关键发现：**

1. **女侠驱动力几乎零方差**（σ=0.0004）。原因：女侠 SECURITY sensitivity=2.0，事件对她的维度冲击是确定性的数值计算（不经过 LLM），相同输入 → 相同输出 → 相同驱动力。这验证了核心论点：**数值层是确定性的，"性格"由参数决定而非随机性。**

2. **老儒生存在微小方差**（σ=0.009）。原因：驱动力计算本身是确定性的，但 LLM 行为生成的 tool calling 结果会写入记忆，记忆内容差异会导致后续轮次维度状态的微小偏移。**方差来源于 LLM 的行为选择，而非数值引擎本身。**

3. **驱动力单调递增**。两个角色从 S0 到 S9 持续上升，因为：(a) 追兵事件持续冲击安全感，(b) 衰减速率（0.03/tick）低于事件冲击频率，(c) 后期无新事件但基线回归尚未完全消化前期积累。这揭示了一个设计需求：**无事件期间需要更长的冷静期让系统回归稳态。**

4. **LLM 行为层展现出可贵的多样性**。虽然驱动力数值高度一致，但 5 轮中角色的具体行为（说什么话、做什么动作）各不相同。老儒生有时选择"挺身而出"，有时选择"虚张声势"；女侠有时选择"挡在前面"，有时选择"喊话引开追兵"。**相同的数值压力，不同的行为表达——这正是"数值涌现"的含义：决定"做不做"的是数值，决定"怎么做"的是 LLM。**

### 3.4 冥修者：三十天的缓慢变化

一个更深层的案例。深山冥修者，闭关三十年。

| 维度 | 值 | baseline | 说明 |
|------|-----|----------|------|
| 好奇心 | 0.9 | 0.9 | 追求真理 |
| 独立性 | 0.95 | 0.95 | 不需要任何人 |
| 依恋需求 | 0.05 | 0.05 | 几乎为零 |
| 释放冲动 | -0.8 | 0.0 | 极致压抑 |
| 情绪波动 | -0.9 | 0.0 | 心如止水 |

**第1天：旅人倒在他洞口。**

```
事件: interpersonal, intensity=0.8, valence=negative
安全感: 0.7 → 0.7 - (0.8 × 0.8 × 1.0 × 0.3) = 0.51  // 警觉
同理心:  0.4 → 0.4 + (0.8 × 0.3 × 0.5) = 0.46        // 微升

驱动力重算: safety=0.52, value=0.63 → max=0.63 > 打断阈值0.5 → 行为涌现
```

他被打断了。不是"决定"去查看，是价值驱动（好奇心+道德准则=0.63）推过了阈值。

**第1-30天：旅人养伤。**

每天产生低强度 interpersonal positive 事件（说话、共食、换药），微小的偏移每次 0.01-0.03，方向一致：

| 维度 | 第1天 | 第15天 | 第30天 | 机制 |
|------|-------|--------|--------|------|
| 依恋需求 | 0.05 | 0.09 | 0.15 | 反复 interpersonal positive → ATTACHMENT 微升 |
| 同理心 | 0.46 | 0.52 | 0.58 | 救人行为反复激活共情 |
| 独立性 | 0.95 | 0.91 | 0.86 | 习惯有人存在 |
| 情绪波动 | -0.9 | -0.7 | -0.5 | 不再完全静止 |

**没有人写"他学会了关心"。是依恋需求从 0.05 涨到了 0.15。**

**第30天：旅人离开。**

```
依恋需求:     0.15 → 0.15 - (0.6×0.3×1.2) = -0.07  // 依恋被撕开
安全感:       0.51 → 0.37                            // 不安
释放冲动:     -0.6 → -0.53                           // 压不住了
```

他不会追出去——独立性还有 0.86，释放冲动才 -0.53。驱动力来自 safety 通道，不是 social。他的"不安"不是思念，是安全感被打回了基线以下。

但冥想时会走神——依恋需求的**基线**已经从 0.05 **迁移**到了 0.08。洞里空了一块。

---

## 4. 讨论

### 4.1 情绪不是变量，是涌现

愤怒不是某个维度叫"愤怒"。愤怒 = 高情绪波动 + 高释放冲动 + 低安全感的组合涌现。恐惧 = 低安全感 + 高情绪波动 + 低风险承受。你不需要定义这些情绪变量，它们会自己长出来。

这意味着：**添加新的情绪不需要改代码。** 只需要新的维度组合模式。

### 4.2 打断是能动性的可观测标志

打断不是预设规则。打断是驱动力超过忍耐阈值后的涌现行为。

证明：
1. 老儒生（情绪波动-0.6, 释放冲动-0.6）→ 几乎不会打断任何人
2. 但如果对方触犯了他的道德准则0.7或触发创伤记忆 → 可能涌现打断
3. 暴躁年轻人（情绪波动0.7, 释放冲动0.6）→ 经常打断

打断的涌现证明角色不是在执行预设对话树，内在驱动可以超越交互规则的约束。

### 4.3 信息不对称创造戏剧

角色只能看到对方的行为输出（语言、动作），看不到对方的内部20维状态和记忆。这和现实一致——你只能看到别人做了什么，不知道他为什么这么做。

叠加感知的物理约束：背对事件的人看不到，远处的声音只能隐约听到。同一个世界，面向不同看到的不同。误解从物理层面就是不可避免的。

### 4.4 人格体系的统一映射

Big Five、MBTI、九型人格、Dark Triad、DISC——全部可以映射到同一个20维空间。已验证26/26人格特质维度覆盖。区别只是数值不同，系统不变。

**没有硬编码的"性格类型"。**

### 4.5 天道：规则的生长

角色可以做任何事——包括规则库里没有的事。天道系统（LLM）介入判断合理性，生成新法则写入库。下次再遇到，直接计算，不再调 LLM。

第一次有人"纵火"→ 天道生成纵火法则（蔓延速度=风力×0.5，伤害=10HP/分钟）→ 入库。第二次纵火 → 直接算。

天道可以关闭。关了之后世界只执行已有规则，未知行为直接拒绝。**这是"有神"和"无神"的区别。**

### 4.6 时间切片：叙事节奏参数化

从子弹时间（3秒/切片）到周级（1周/切片），12级粒度。LLM 能感知到"现在是慢镜头"还是"一天过去了"——时间粒度本身也是信息。

## 5. 讨论

### 5.1 Baseline 验证了什么

5 轮 × 10 切片的 baseline 实验验证了以下论点：

**数值层确定性：** 女侠驱动力跨轮标准差仅 0.0004，证明数值引擎在相同输入下产生完全相同的输出。性格不是随机性，是参数。

**行为层多样性：** 相同的驱动力数值，LLM 产生了不同的行为选择。这验证了"数值决定做不做，LLM 决定怎么做"的分工。

**驱动力单调递增问题：** 无事件期间系统未能回归稳态，揭示了当前衰减参数（0.03/tick）在高频事件场景下不够用。这是可调参数，不是架构缺陷。

**角色区分度：** 老儒生（均值 0.804）和女侠（均值 0.817）的驱动力存在稳定差距，且方差模式不同（老儒生 σ=0.031 vs 女侠 σ=0.017），说明不同性格参数确实产生了可区分的行为模式。

### 5.2 行为通性：5轮中的稳定模式

尽管 LLM 每轮输出不同，5 轮中出现了高度稳定的行为模式——这些不是预设的对话树，而是从性格参数中涌现的"角色人格"。

**老儒生的行为通性：**

| 切片阶段 | 稳定行为模式 | 涌现来源 |
|----------|-------------|----------|
| S0-1（门板碎裂） | "放下毛笔，缓缓起身" → "轻手轻脚走向门边" → 问"何人造次" | 情绪波动-0.6 → 不慌不忙；务实性0.6 → 先观察再行动 |
| S2-3（侦查期） | "从抽屉取出短剑"或"拿起柴火棍" → 透过门缝窥视 → 保持安全距离 | 独立性0.8 → 自己处理；风险承受-0.5 → 不冒险靠近 |
| S4-5（追兵对峙） | "挺直腰板，手持短剑/柴火棍" → "双手作揖" → 以礼相劝但据理力争 | 道德准则0.7 → 不能见死不救；务实性0.6 → 先礼后兵 |
| S6-9（持续对峙） | 反复"整理衣冠，挺直腰板，双手作揖" → 以儒家大义震慑 → 拖延时间 | 掌控欲-0.3 → 不追求控制局面；自尊心0.2 → 不怯场但不傲慢 |

**5轮老儒生说话的共同模式：** 几乎每轮都出现"老夫隐居于此，素来与人无争"、"尔等何人，竟敢擅闯"、"以礼相待"等措辞。这不是模板——记忆层的文化背景设为"书香门第，说话带文言气"，LLM 据此自发选择了文言措辞。

**女侠的行为通性：**

| 切片阶段 | 稳定行为模式 | 涌现来源 |
|----------|-------------|----------|
| S0（破门进入） | "迅速移动到角落阴影处" → "右手按剑柄" → "左手按住伤口" | 风险承受0.8 → 立刻战斗姿态；安全感sensitivity=2.0 → 极端警觉 |
| S1-2（侦查期） | "屏住呼吸，紧贴墙壁" → 透过缝隙观察 → 同时撕衣襟包扎伤口 | 独立性0.9 → 自己处理伤口不依赖他人；务实性0.7 → 先自救再考虑他人 |
| S3（马蹄声） | "忍痛冲出阴影，站到老儒生身前" → "老先生小心！外面有马队！" | 同理心0.2（低）但依恋需求被激活 → 对救命恩人的责任感涌现 |
| S4-9（追兵对峙） | **反复出现的核心模式**："站到老儒生身前挡住" → "你们要找的人是我" → "别为难老人家" | 亲和度-0.2 → 对追兵毫不客气；道德准则0.3 → 恩怨分明的江湖义气 |

**5轮女侠说话的共同模式：** 几乎每轮都出现"你们要找的人是我"、"有种冲我来"、"别为难老人家"三句话。核心逻辑一致：承认自己是目标，要求不连累恩人，准备死战。措辞各不相同但意图完全一致。

**最有说服力的通性发现：**

**"谁保护谁"的角色分化完全稳定。** 5轮中，每当追兵威胁升级：
- 老儒生**每次**都选择"挺直腰板站在前方"——用威严和言语对抗（文人方式）
- 女侠**每次**都选择"冲到老儒生身前挡住"——用身体和剑对抗（武人方式）

两个角色的"保护方式"完全由其参数决定：老儒生务实性0.6（理性应对）+ 道德准则0.7（不能不管）；女侠独立性0.9（自己扛）+ 风险承受0.8（不怕死）+ 释放冲动0.5（压不住）。**没有人预设"文人用嘴，武人用剑"——这是参数涌现的结果。**

### 5.2 当前局限

- Baseline 仅覆盖追兵场景（高强度连续事件），未测试长期低强度场景（如冥修者30天案例）
- 因果引擎仅支持两层传播，深层因果链可能需要图搜索
- 感知系统目前为 2D，未考虑三维空间和遮挡关系
- 持续行为系统（事件链+补间动画）仍在设计中
- LLM 行为的"质量"未做系统评估（如一致性、角色保持度）

### 5.3 待验证假设

- **基线迁移的稳定性：** 重大事件能否永久改变基线？迁移后的系统能否找到新的稳态？
- **多角色涌现的复杂性：** 3个以上角色同时交互时，因果网络是否会产生不可预测的震荡？
- **天道系统的规则收敛性：** 规则库是否会无限膨胀？是否存在自然收敛的机制？
- **衰减参数的校准：** 不同场景节奏下，最优衰减速率是否有规律可循？

### 5.3 愿景

一句话：我们在做一个**有灵魂的世界引擎**。

不是更好的对话机器人，不是更逼真的 NPC。是一个你把角色扔进去，然后退后一步看他们会做什么的世界。
